
# Dataset

The datasets used in this project were obtained from the ENCODE Project.

Due to GitHub storage limitations, raw genomic datasets are not included in this repository.

Files required:

- chr22.fa
- ENCFF946OGO.bed
- ENCFF960TEU.bed
- ENCFF289LLT.bed
- ENCFF864OSZ.bed
- ENCFF256ZMD.bedpe

Download sources:

https://www.encodeproject.org/

Place all downloaded files inside the data/ directory before running the pipeline.
